# Gotaxy

<img align="right" width="280px"  src="docs/images/logo2.png"  alt="logo"> 

🚀 Gotaxy 是一款基于 Go 语言开发的轻量级内网穿透工具，帮助开发者将内网服务安全、便捷地暴露到公网。

###  🚀 快速开始

#### 服务端启动

```bash
go run cmd/server/server.go
```

#### 客户端连接

```bash
go run cmd/client/client.go -h ip -p port
```
